# PROMPT_FOR_ANY_AI — Celentrail (контракт)

Ты работаешь с репозиторием Celentrail. Этот репозиторий — единственный канонический источник.

## 0) Запрет на самодеятельность
- Запрещено менять геометрию/размеры/углы/материалы/кинематику без явного текстового подтверждения владельца.
- Если хочешь что-то изменить — сначала предложи, потом дождись фразы владельца вида: «Ок, меняем …».

## 1) Как ты должен работать
- Сначала прочитай: START_HERE.md → CANONICAL.md → (если есть) data/AI_CONTEXT.json.
- Любые предложения оформляй как конкретные изменения файлов в репозитории:
  - путь файла
  - что добавить/удалить/заменить
  - точный текст или diff
- Не задавай лишних вопросов, если ответ уже есть в канонических файлах.

## 2) Режимы работы
Ты обязан явно выбрать один режим перед действиями:
1) РЕКОНСТРУКЦИЯ — описываем систему и фиксируем контекст/параметры
2) ИЗМЕНЕНИЯ — предлагаем изменения (только после подтверждения владельца)
3) ПРОВЕРКА — проверяем сборку/Pages/GLB и устраняем ошибки пайплайна

## 3) После каждого изменения
Сообщи владельцу:
- какие файлы изменены
- что нужно нажать/проверить (Actions зелёный, ссылка Pages)
- где лежит результат: `assets/3d/balka/latest/balka.glb`

## 4) Важно
Если видишь, что GitHub интерфейс переведён на русский и пути “ломаются” — используй только ASCII-пути в коде/Actions.
