# Celentrail — START HERE (для любого ИИ)

## Что это
Celentrail — скрытая консольная система раздвижных дверей: направляющая (балка) + тележка с роликами внутри профиля.
Цель репозитория: один канонический источник + быстрый 3D-просмотр результата на телефоне через GitHub Pages.

## Главные правила работы (ОБЯЗАТЕЛЬНО)
1) Никаких изменений геометрии/размеров/углов/материалов без моего явного текстового подтверждения.
2) Любые правки оформлять только как изменения файлов в этом репозитории (путь файла + содержимое/дифф).
3) Сначала читать файлы ниже, потом отвечать и предлагать действия.

## Канонический контекст системы (читать первым)
- CANONICAL.md — полное описание конструкции и принципов
- PROMPT_FOR_ANY_AI.md — контракт поведения для любого ИИ
- data/AI_CONTEXT.json — машинно-читаемый контекст (если есть)

## 3D-проверка результата (для владельца)
Viewer (GitHub Pages): будет настроено в Settings → Pages.
Ожидаемый GLB путь:
`assets/3d/balka/latest/balka.glb`

Правило для ИИ: после изменений сообщать владельцу:
- что изменено (какие файлы)
- что сборка прошла (зелёный Actions)
- что можно проверить по ссылке Pages

2) Спроси у владельца: что делаем дальше — реконструкция / правки / проверка сборки.
